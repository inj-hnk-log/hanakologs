# 記録

タグ: #異質脳, 警鐘

思考：

```
温かい飲食物を摂ることが神経刺激になっており、「温める＝癒し」ではなく「温める＝危険回避要件」になっていると実感。味噌汁すら調整しないと神経が燃える。

気温が涼しい日なら熱い飲食もどうにか耐えられるが、元来高温には弱い。
特に先週から神経刺激の閾値が下がり、歯の神経の過敏が極端に増幅されている。
```

感覚：

```
90℃前後の通常の味噌汁温度では過刺激。
飲用には黄連解毒湯を要するレベルの神経過敏。
現在は人肌（40℃程度）で調整し、かろうじて摂取できている。

歯の神経痛が顕著で、わずかな温度や刺激でも過剰反応が起こっていた。
食事内容も物理的・神経的刺激を避ける構成に移行。
```

工夫：

```
味噌を溶く際、90℃ではなく40℃程度のぬるま湯を使用。
温かさよりも「燃えない温度」が優先されるため、飲用のための“温度最適化”が必須。

夕食はシリアル＋パン＋豆乳で構成。加熱・調理の必要がなく、刺激も少ない。栄養面では簡易だが、現状の感覚過敏には妥当な対応。
```

⌘（気づき）：

```
味噌汁が“戦うための食”ではなく、“生きるための点滴”に変化している。
神経刺激を避けつつ必要な塩分と発酵成分を摂るための最小限のライフライン。

外的条件（気温）により許容できる飲食温度が変化すること、そして内部状態（神経刺激の閾値）が恒常的な食習慣の変更を促す点に注意。身体的なトリガーが、食生活・行動に波及する構造を再認識。
```
