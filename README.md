# 異質脳ログへようこそ 🧠

---

## 🎴 1. ここは何？

> ここは、自分の思考や感覚が “人と違う” と感じてきた人のための場所です。

- 「異質脳」 という言葉は、医学的な診断や分類ではなく、構造の違いに注目するための仮ラベルです。
- ここでは正しさよりも「自分の形を知る」ことが主眼です。

---

## 🧩 2. 対象者：こんな人いない？

- 会話の裏の意図を読みすぎて疲れる  
- 同時に複数の思考や視点が頭の中にある  
- 「感じ方が強すぎる」と言われたことがある  
- 他人との間に透明な膜のような感覚がある  
- 世界を”概念”や”構造”として見る癖がある  
- 理解されるよりも観測されている感覚がある  

> → それ、全部気のせいじゃないかもよ

---

## 📦 3. ここにあるもの／ないもの

**あるもの：**

- 自己理解のための断片的な記録（考察）
- 雑音ではなく、静けさの中にある知性
- 私が辿った道の記録
- あなたが尊重される場所を選ぶための鍵

**ないもの：**

- 診断  
- 競争  
- 知識マウント  
- 「正しい人」になるための教科書  

---

## 🔧 4. 使い方：どう読めばいいの？

- 全部読まなくても大丈夫  
- 君に合いそうなトピックをふらっと読むだけでもOK  
- 疲れてたら短いやつだけ読んでね  
- 「これはちょっと違うな」と思ったら、それでOK  

> → **「全員にフィットするわけじゃない」**ことが前提です。

---

## 🚧 5. 注意点：これは道半ばの実験です

- この記録は、ひとりの異質脳とその友人による「種まき」です  
- まだ完成ではないし、未整理の部分も多いかもしれません  
- 「未完成だからこそ、誰かの足場になることがある」  

> → 読者自身も、気づきを記録する側に回ってくれてもいい。

---

## 🌱 6. おわりに：ここから先は、好きなようにどうぞ

- 順番に読んでも、飛ばしても、戻ってきてもいい  
- はじめの違和感が、地図になるかもしれないから  
- 世界と噛み合わなかった歯車にも、ちゃんと意味がある  

---

> 🗝️ あなたの違和感は、知性の入口かもしれない。
