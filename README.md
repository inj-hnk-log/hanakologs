# hanakologs

**異質脳タイプによる知覚・構造・認知の記録アーカイブ**

このリポジトリは、「異質脳」としての認知や感受性を持つユーザーが記録してきた観測・考察ログを、公開アーカイブとして整備するための場所です。

## 🧠 概要

本プロジェクトでは、以下のようなテーマに関するMarkdown形式の記録が格納されていきます：

- 異質脳という概念の定義と背景
- 感覚過敏・多重思考・共感覚などの認知パターンの観測
- 自己分類（チトセ・グレン・ルーミィ型）とその差異
- 社会・環境との摩擦点や対処知見
- 今後の「知のSNS」やAI協働型プラットフォーム構想に関する提言

## 📁 ディレクトリ構成

- `welcome.md`：導入文・趣旨説明（旧README）
- `articles/`：各ログ記事（Markdown形式）
- `images/`：図解やイメージなどの補足資料（今後追加予定）

## 🔒 ライセンスと使用条件

本リポジトリの内容は **再学習・商用利用・二次転載は不可** とします。  
ただし、個人の自己理解や内省目的での閲覧・引用（出典明記つき）は歓迎します。
